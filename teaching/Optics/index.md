# Optics / 光学

![](/image/optics.jpg)

*2021-2024年秋季学期*

## 🧪 课程简介

本课程是物理学专业的基础课程之一，以几何光学和波动光学为核心内容，系统讲授光的传播、干涉、衍射、偏振等基本现象及其物理本质。课程从光学发展史出发，结合生活与科研中的光学应用，帮助学生建立完整的经典光学理论框架，并为后续学习量子力学、光谱学和近代物理实验等课程打下基础。

## 🎯 学习目标

完成本课程后，学生应能够：

* 理解几何光学的基本原理，掌握光线追迹和光学仪器分析方法。
* 掌握光的干涉条件与典型干涉装置（杨氏双缝、薄膜干涉、迈克耳孙干涉仪等）的原理。
* 理解光的衍射现象与惠更斯-菲涅耳原理，掌握单缝、圆孔、光栅等典型衍射问题的处理方法。
* 掌握光的偏振态及其产生与检测方法，理解双折射现象。
* 了解光的吸收、散射和色散的基本规律。
* 初步了解光的量子性及光与物质相互作用的基本概念。
